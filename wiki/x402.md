# x402 協議

基於 HTTP 402 狀態碼的鏈上微支付標準，由 Coinbase 主導開發後移交 Linux Foundation，Google、Stripe、Visa、AWS、Mastercard 為創始成員。讓 AI Agent 可在鏈上直接付費購買服務——Agent to Agent 的金融基礎層。

## Current Status

- 上線 **15 天突破 100 萬筆交易**，[[virtuals|Virtuals]] 一家貢獻 **99% volume**（as of 2026-04-24）
- Virtuals：18,000 個活躍 Agent，每季 170 萬個任務，月收費 **$12-15 萬美元**
- 已移交 **Linux Foundation**，創始成員：Google、Stripe、Visa、AWS、Mastercard、Solana Foundation
- 早期 Circle 版本（2026-03/04）已累計 1.4 億筆微支付（9 個月，均值 $0.31）
