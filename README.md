# boba-wiki

LLM-maintained knowledge base built from [BOBA Daily](https://t.me/test3635) reports.

- **raw/** — immutable daily report archive
- **wiki/** — entity pages maintained by Claude (the knowledge base)
- **schema/** — rules that govern how the LLM operates on the wiki
- **index.md** — catalog of all wiki pages
- **log.md** — chronological operation log

Open this repo as an Obsidian vault to browse the wiki with graph view and `[[wiki-links]]`.
