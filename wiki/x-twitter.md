# X（前 Twitter）

Elon Musk 旗下社群媒體平台，已宣示跨入金融服務。2026-04-15 在美國和加拿大 iPhone 正式上線 Cashtags 功能，社群平台首次正式成為交易終端。

## Current Status

- **Cashtags** 在美國和加拿大 iPhone 上線：搜索 $BTC、$AAPL 或貼合約地址，直接顯示即時價格圖表 + 相關討論（as of 2026-04-15）
- 加拿大用戶可透過 **Wealthsimple** 試點直接在 X 內交易
- 每天有數十億美元的資金配置決策取決於 X 上的內容（產品負責人 Nikita Bier 語）

## Key Events

- **2026-04-15** — 美國和加拿大 iPhone 上線 Cashtags：搜 $BTC/$AAPL 或貼合約地址顯示即時圖表與討論。加拿大透過 Wealthsimple 試點 X 內交易。Nikita Bier（X 產品負責人）：「每天有數十億美元的資金配置取決於人們在 X 上看到的內容。」社群平台正式跨入金融服務。Source: [[raw/2026/04/15]]
