# OpenClaw

開源 AI 模型，CNBC 稱其爆紅速度為「ChatGPT 時刻」。便宜的開源模型能做到與昂貴 API 差不多的效果，引發業界擔憂 AI 正在商品化。Anthropic 後來調整 API 政策試圖限制其整合。

## Current Status

- 已觸發「AI 商品化」討論（as of 2026-03-22）
- Anthropic 已封殺 OpenClaw 第三方整合，改 API 按量計費（2026-04-05）
- 開發者成本可能從 $20/月飆到上千

## Key Events

- **2026-04-05** — Anthropic 封殺 OpenClaw 與 Claude API 的第三方整合，改為按量計費。開發者成本大幅上升，引發社群反彈。Source: [[raw/2026/04/05]]
- **2026-03-22** — CNBC 報導稱 OpenClaw 爆紅速度令業界擔憂「AI 模型正在商品化」，與 Qwen3-Coder 效率突破同時出現，共同壓縮大廠 API 定價空間。Source: [[raw/2026/03/22]]
